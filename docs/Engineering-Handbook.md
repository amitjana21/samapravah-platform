# Sama Pravah Engineering Handbook

## Project Overview

Sama Pravah is a home services platform connecting customers with trusted technicians for services like:

- Electrical
- Plumbing
- AC Repair
- Cleaning
- Carpenter
- CCTV
- Appliance Repair

The platform consists of:

- Customer Website
- Android Mobile App
- Admin Dashboard
- Firebase Backend
- Future Technician Portal

# Coding Standards

Use Functional Components

Use React Hooks

Avoid duplicate code

Prefer reusable components

Keep files under 300 lines when possible

Never hardcode secrets

Never break existing routing

Preserve mobile app compatibility

Desktop website and mobile app should coexist.
